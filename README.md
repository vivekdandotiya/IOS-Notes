# QuickNotes ✨ - Modern iOS Note Taking

![QuickNotes Poster](./Images/Notes-Poster.jpg)

QuickNotes is a clean, efficient, and visually stunning note-taking application designed for the modern iOS ecosystem. Built with a focus on simplicity and a premium user experience, it allows you to capture your thoughts instantly and manage them with ease.

## 🚀 Key Features

- **✨ Modern & Minimalist UI**: A carefully crafted interface featuring a card-style layout, premium typography, and subtle animations.
- **⚡ Fast & Fluid**: Optimized for speed, ensuring a seamless experience from the moment you open the app.
- **💾 Robust Persistence**: Leveraging **Core Data** for reliable, localized storage of all your important thoughts.
- **🌓 Adaptive Theme**: Full support for both Light and Dark modes, beautifully adjusting to your system settings.
- **📱 Universal Design**: A truly responsive experience that scales perfectly across both **iPhone** and **iPad**.
- **🔍 Smart Search**: Instantly find any note with powerful real-time search capabilities.

## 🛠️ Technical Stack

- **SwiftUI**: Modern, declarative UI framework for a responsive and interactive experience.
- **Core Data**: Powerful local data persistence and management.
- **MVVM Architecture**: Clean, scalable, and maintainable code structure.
- **SF Symbols**: Utilizing Apple's extensive iconography for a native and intuitive look.

## 💡 Motivation

The vision for QuickNotes was to create a note-taking experience that feels personal and premium. Instead of just another utility app, QuickNotes was developed to explore the boundaries of SwiftUI's design capabilities, resulting in an interface that is as beautiful as it is functional. This project represents a journey into creating high-quality, production-ready iOS software that prioritizes the user's needs.
